Prerequisites:

	• Java 17+
	• Maven
	• Kafka (for messaging features)
	• PostgreSQL

Installation Steps:
	
	1. Clone the repository:
		git clone https://github.com/nicholastimothycoffman/AustinAutoAdjust

	2. Navigate to the project directory:
		cd AustinAutoAdjust

	3. Build the project:
		mvn clean install

	4. Start the application:
		mvn spring-boot:run
